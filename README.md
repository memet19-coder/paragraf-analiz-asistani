# Paragraf Analiz Asistanı

Türkçe paragraf çözümleme ve dil bilgisi analizi için hazırlanmış React + Node.js web uygulaması.

## Çalıştırma

1. `start-app.cmd` dosyasını çalıştır.
2. Tarayıcıda `http://127.0.0.1:5173` adresi açılır.
3. Paragrafı yazıp `Analiz Et` düğmesine bas.

## Özellikler

- Modern ve mobil uyumlu Türkçe arayüz
- Büyük paragraf giriş alanı
- Kartlar halinde paragraf analizi
- Sekmeli sonuç ekranı
- Dil bilgisi analizi tablosu
- Konu, ana fikir, yardımcı fikirler, özet ve başlık önerileri
- İsim, sıfat, zamir, zarf, edat, bağlaç, çekimli fiil ve fiilimsi tespiti

## Teknik yapı

- Frontend: React + Vite
- Backend: Node.js + Express
- Tasarım: Tailwind CSS

Geliştirme için:

```bash
pnpm install
pnpm run dev
```
