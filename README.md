# Paragraf Soru Asistanı

Türkçe paragraflardan öğrenciler için çoktan seçmeli soru seti hazırlayan React + Node.js web uygulaması.

## Çalıştırma

1. `start-app.cmd` dosyasını çalıştır.
2. Tarayıcıda `http://127.0.0.1:5173` adresi açılır.
3. Paragrafı yazıp `Soruları Hazırla` düğmesine bas.

## Özellikler

- Modern ve mobil uyumlu Türkçe arayüz
- Büyük paragraf giriş alanı
- Konu, ana fikir, çıkarım, söylenemez, çıkarılamaz ve anlatım biçimi soruları
- Her soru için dört seçenek
- Her soru için `Cevabı göster` düğmesi
- Doğru cevap ve kısa açıklama alanı
- Öğretmen ve öğrenci kullanımına uygun sade ekran

## Teknik yapı

- Frontend: React + Vite
- Backend: Node.js + Express
- Tasarım: Tailwind CSS

Geliştirme için:

```bash
pnpm install
pnpm run dev
```
